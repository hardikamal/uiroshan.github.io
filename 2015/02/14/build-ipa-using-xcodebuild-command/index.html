Build IPA using XCodeBuild Command

Alternative method to build IPA without XCRun command.

Posted by Upul Abayagunawardhana on February 14, 2015

There will be plenty of posts about the same topic can be found on the internet. But I am writing it again, just to make a note for myself. Also another thing I noticed, most of the posts are using xcrun command to package IPA. But XCodebuild also has ability to package as an IPA.

There are two steps to build an IPA. First we have to archive our project.

  • If you are using workspace you will have to provide workspace and sheme as inputs to the xcodebuild command. If you are using one xcode project with multiple targets, specifying target will be enough.
  • we can specify the path to store final out put archive file as archivePath.
xcodebuild -workspace MySimpleApp.xcworkspace/ -scheme Simple -archivePath ../archives/simple  clean archive

Next we have to export our archive into IPA. For this we have to provide export format as IPA, archive path and export path.

  • exportProvisioningProfile option Specifies the provisioning profile that should be used when exporting the archive. Name of the provisioning profile can be found from the developer portal
xcodebuild -exportArchive -exportFormat IPA -archivePath ../archives/simple.xcarchive -exportPath Tobys.ipa -exportProvisioningProfile 'XC Ad Hoc: io.github.uiroshan'
